    📖 使用说明

    🎯 这个工具是做什么的？

    这是一个电影排名投票工具。你可以把自己豆瓣看过的电影导入进来，然后通过一轮一轮的「二选一」投票，让 ELO 算法帮你算出你最爱的电影排行榜。

    核心思路很简单：两部电影同时出现，你选更喜欢的那一部。选得越多，算法越能精准判断你的口味。

    📥 第一步：导入豆瓣数据

    1在「导入数据」页面，点击或拖拽你的豆瓣已看电影 JSON 文件到虚线框里。

    2勾选你想参与排名的星级（默认 4~5 星）。如果只想让五星电影参与排名，就只勾选五星。

    3点击「导入已选数据」，页面顶部会显示导入成功提示。

    提示：文件从哪里来？豆瓣已看电影页面导出 JSON，或使用浏览器插件（如「豆瓣备份助手」）导出的电影数据。只要文件里是电影数组，工具一般都能识别。

    ⚔️ 第二步：开始 1v1 对比

    切换到「1v1 对比」页面，你会看到两张电影卡片。规则如下：

    • 点击你更喜欢的那张卡片，它就是本轮赢家。
    • 如果两部都喜欢、难以抉择，点击「平局」。
    • 如果暂时不想判断，点击「换一批」跳过。
    • 顶部进度条会显示你已完成的对决数。对决越多，排名越稳定。
    小技巧：不用纠结「谁排第一」。只要凭直觉反复选择，ELO 算法会自动把真正喜欢的电影推到前列。

    🏆 第三步：查看排行榜

    切换到「排行榜」页面，可以看到当前排名。你可以选择 TOP 10 / 20 / 50 / 100，并点击「导出长图」生成一张 PNG 图片，方便分享到朋友圈或社交媒体。

    排行榜只显示「排名 + 电影名」，前三名会用金色、银色、铜色高亮，简洁美观。

    💾 数据如何保存？

    所有投票进度和排名都会自动保存在当前浏览器的本地存储里。也就是说：

    • 刷新页面不会丢数据。
    • 关闭浏览器再打开，仍然可以继续投票。

    如果想换设备或备份进度，可以点击「导出进度」保存一个 JSON 文件；换设备后在新设备上用「导入进度」恢复即可。

    🔄 常见问题

    Q：导入时提示「未识别到电影记录」怎么办？
    请检查 JSON 文件是否包含 items、subjects 或 interests 字段。如果文件结构很特殊，可以尝试用常见的豆瓣导出工具重新导出。

    Q：我想重新开始怎么办？
    在「导入数据」页面点击「重置数据」，会清空所有电影和投票记录。注意此操作不可恢复。

    Q：排行榜可以导出高清图片吗？
    点击「导出长图」后会生成 PNG 并自动下载。图片宽度固定为 520px，适合手机查看和分享。
